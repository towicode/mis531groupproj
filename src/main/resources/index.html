Query 1

It's important for the company to keep track of birthdays for company morale. Company policy is that every Friday at every branch they have a birthday party for all who had a birthday that week. Corporate also needs to be aware so they can send cards to the employee whose birthday it is. This query returns a complete list of every Birthday that will happen this week. Monday -> Sunday. For example, if ran on Wednesday it will still return Monday -> Sunday. It will also give the employees name, dob, and the branch they work at.

Query 2

The company ensures that each client site is visited so often based upon its size. If the size of the location is > 1000 then once a week, if the size is > 500 they will need a visit twice a month. Otherwise, they will just need a visit once a month. This query will check and list what sites are due for a visit. It will list them in urgent order. This query will also list the client rep team leader that last visited them so they can be made aware.

Query 3

Find the client id, client name, order ship date, order status, client rep that handled the order, quantity of order and price of the product. Display only orders where status is not null, quantity is not null, and the price is greater than 500. Order by quantity in desc order.

Query 4

Determine our top salaried employee where the start date is >= 01-Jan-2000. Order them based on there on salary in desc order. Show if they have achieved a degree.

Query 5

It is important for managers to track and review the top 5 Customer Service Representatives who has the most incident reviews. Using this information, this can be reviewed for performance reason and discuss with the employees. Display top 5 Customer Service Representatives who has the most Incident Review. For details, display their first name, last name, number of incidents, and rank for review purpose.

Query 6

Display courses that is not completed by employee students so that the manager can remind them to complete course. Display student's full name, course number, course name, and class completion status and order by student’s name.

Query 7

Determine the product groups that each rep represents and make sure the overlap between any two product-groups is below 2/3rds (i.e. 67%) and ensure that there exist at least ten representatives for each group.

Query 8

Find all client sites that we consider large based on their size and find reps that live in the same state and display there specialty areas. This will show us the reps that we can assign to each client site and their specialty areas

Query 9

For new reps we need to assign them between ten and up to twenty different clients within a region to visit during their first year on the job. So for each new rep we need to determine what region their branch is a part of and then find clients that are in that region.

Query 10

It's important for the company to find which product lines and products with the highest number of orders and then rank them from highest to lowest. This to identify promotions/discounts and target marketing campaigns
